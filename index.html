img_hosting — GitHub Pages

This repository serves static images via GitHub Pages. To use an image in Gmail (or a signature), use the GitHub Pages URL:

https://SamiJoel.github.io/img_hosting/images/placeholder.svg

Drop your images into the images/ folder on the main branch. After pushing, the public URL for an image named image.png will be:

https://SamiJoel.github.io/img_hosting/images/image.png
